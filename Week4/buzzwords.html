Hoisting
    Declaring a variable anywhere in the code
    is equivalent to declaring it at the top.
    This also means that a variable can appear
    to be used before it's declared.
    This behavior is called "hoisting".

Strict Mode
    An inert string added to the top oof your javascript file that indicates
    that you want editors and interpreters to remove the looseness and flexibility of javascript
    that can cause your code to behave in unexpected ways.

    // Whole-script strict mode syntax
    "use strict";

    https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Strict_mode


Semaphore
    A flag for signalling.
    Usually a boolean or an interger/counter,
    used to start and stop a resource consuming process,
    based on the relied upon resources availability.
