Transpiler:
        Similar to a *compiler* new-fangled non-standard dialects of css, javascript, or html;
        to real production classic standard versions of css, javascript, and html.

        Examples:
            CSS:        LESS, SASS
            JAVASCRIPT: CoffeeScript, TypeScript
            HTML:       JADE, Emmet

    Compiler:
        Translates source code that is written by programmers for programmers;
        to machine code that uses the atomic operations that a computer's CPU is capable of.

        Compiled langueges:
            C, C++
        Compiled to virtual machine:
            Java, C#
